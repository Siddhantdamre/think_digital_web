# think_digital_web

Student/community web platform prototype with event pages, member management, MongoDB models, and Node backend.

## What It Does

- Provides pages for events, blogs, members, resources, contact, and login.
- Includes Node/Express-style backend experiments.
- Defines data models for members and events.

## Tech

JavaScript, Node.js, MongoDB, HTML, CSS.

## Repository Map

- `server.js`, `backupsever.js` - backend experiments.
- `Event.js`, `Member.js` - data models.
- `index.html`, `about.html`, `events.html`, `members.html`, `resources.html` - UI pages.

## Status

Prototype web platform.
